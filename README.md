# sandbox-poinst-calc

### To launch:
1) clone repo
2) Run ```npm install``` in ```sandbox-poinst-calc``` (root folder)
3) Run ```npm run presetup```
4) Run ```npm run dev```


Created with CRA;  
State management - MobX;  
UI library - AndD;  
Backend - NodeJs + express + concurrently;
